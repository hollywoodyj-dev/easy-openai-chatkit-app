# Rename Vercel project to wisewave-chatkit

1. Open **Project Settings** (General):
   - **Link:** https://vercel.com/jing-yangs-projects-db5d1ce8/easy-openai-chatkit-app/settings/general

2. Under **Project Name**, change:
   - From: `easy-openai-chatkit-app` (or `easy-openai-chatkit-app-two`)
   - To: **wisewave-chatkit**

3. Save. Your production URL will become:
   - **https://wisewave-chatkit.vercel.app** (or under your team subdomain)

4. If you use the OpenAI domain allowlist, update it to include the new URL.
